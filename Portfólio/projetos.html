Banco Digital

set de 2024 – set de 2024

Função: Desenvolvedor Java Júnior.

O Banco Digital é um sistema bancário desenvolvido em Java, onde é possível realizar operações bancárias simples diretamente pelo terminal. O projeto simula um banco, permitindo ao usuário criar contas bancárias, fazer depósitos, saques, transferências e listar as contas cadastradas. O sistema é totalmente orientado a objetos, com classes responsáveis por cada operação e para a gestão dos dados do usuário.

Funcionalidades Principais:

Criar Conta: Permite a criação de contas bancárias associadas a uma pessoa (nome, CPF, e-mail), com um número de conta gerado automaticamente. Depositar: Permite o depósito de valores em contas bancárias, atualizando o saldo de acordo com o valor depositado. Sacar: Permite realizar saques, verificando se o saldo da conta é suficiente para a operação. Transferir: Realiza transferências entre diferentes contas, debitando o valor da conta de origem e creditando na conta de destino. Listar Contas: Exibe todas as contas bancárias cadastradas, mostrando informações como nome do titular, CPF, saldo e número da conta.

Tecnologias Utilizadas:

Java 17 para o desenvolvimento do sistema. ArrayList para armazenar as contas bancárias de forma dinâmica. Scanner para capturar entradas do usuário no terminal. Estruturas de Controle de Fluxo (if, loops, switch case) para gerenciar as operações.

Objetivo do Projeto:

O Banco Digital visa demonstrar minha habilidade em desenvolvimento orientado a objetos, manipulação de dados, controle de fluxo e implementação de operações bancárias simples. O sistema foi projetado de forma modular, utilizando boas práticas de programação e abordando aspectos de design como encapsulamento e abstração.




Loja Central Casa

jul de 2020 – ago de 2020

Função: Analista de E-commerce.

O projeto Loja Central Casa foi uma loja virtual dedicada ao comércio de móveis e artigos de decoração, em parceria com dois sócios. Como Analista de E-commerce, fui responsável por várias atividades operacionais e de suporte que garantiram o bom funcionamento da plataforma e a satisfação dos clientes.

Minhas Principais Atividades:

Gerenciamento de Clientes: Desenvolvi e gerenciei uma planilha customizada para acompanhar de perto os pedidos, status de produtos e pagamentos. Isso garantiu um controle eficiente e ágil de todas as operações e uma melhor comunicação com os clientes. Suporte ao Cliente: Prestei atendimento direto aos clientes via e-mail e WhatsApp, resolvendo dúvidas e fornecendo informações detalhadas sobre os produtos adquiridos. Atualizações e Personalização do Site: Fui responsável pelas alterações no site da loja virtual, incluindo a personalização de páginas, atualizações de produtos e ajustes necessários na plataforma Shopify para melhorar a experiência do usuário e manter o catálogo de produtos sempre atualizado.

Tecnologias e Ferramentas Utilizadas:

Excel/Google Sheets: Ferramentas utilizadas para criar e gerenciar a planilha de controle de pedidos, status dos produtos e pagamentos. WhatsApp e E-mail: Para realizar o suporte ao cliente e garantir uma comunicação eficiente.

Objetivo do Projeto: O Loja Central Casa me proporcionou uma excelente experiência em gerenciamento de e-commerce, atendendo diretamente às necessidades dos clientes e realizando a personalização do site para otimizar a navegação e as vendas. Esse projeto me ajudou a desenvolver habilidades práticas em suporte ao cliente, gestão de vendas e administração de plataformas de e-commerce, além de me ensinar a importância de uma comunicação eficiente com os clientes para garantir uma experiência de compra satisfatória.




Projeto ERS

abr de 2020 – jun de 2020

Função: Web Designer (Wix).

O Projeto ERS envolveu a criação de um site de vitrine para uma empresa especializada na venda de barcos, com o objetivo de proporcionar aos clientes uma forma simples e eficiente de acessar os produtos e entrar em contato com os vendedores. O site foi desenvolvido na plataforma Wix, utilizando suas ferramentas de construção e personalização para garantir uma navegação fluída e uma apresentação visual atraente dos produtos.

Minhas Principais Atividades:

Desenvolvimento no Wix: Utilizei o Wix para criar a estrutura do site, incluindo a modelagem das páginas e integração das funcionalidades. Trabalhei com recursos nativos da plataforma para garantir que o site fosse responsivo e fácil de atualizar. Modelagem e Design: Desenvolvi o layout visual do site, escolhendo templates e ajustando o design para criar uma vitrine de produtos que fosse ao mesmo tempo moderna e funcional. Garanti que a navegação fosse intuitiva, facilitando a busca pelos produtos e a interação com os vendedores. Criação de Funcionalidades Interativas: Implementei formulários de contato, integração com ferramentas de WhatsApp e outros meios de comunicação para permitir que os clientes tirassem dúvidas diretamente com os vendedores. Testes e Otimização: Testei o site em diversos dispositivos e navegadores para garantir que a experiência fosse consistente e otimizada, assegurando que os clientes tivessem um acesso rápido e sem falhas aos produtos.

Tecnologias e Ferramentas Utilizadas:

Wix: Plataforma de construção de sites utilizada para o desenvolvimento do projeto, aproveitando suas funcionalidades de arrastar e soltar para criar páginas personalizadas e responsivas. Editor de Wix: Utilização do editor de Wix para design de página, otimização de SEO e personalização de templates. Integração com WhatsApp: Configuração de botões e formulários para facilitar o contato direto entre clientes e vendedores.
